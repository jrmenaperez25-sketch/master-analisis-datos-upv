# Grado en Matemáticas — Proyectos Académicos

Repositorio con los proyectos más destacados del **Grado en Matemáticas**, incluyendo el Trabajo de Fin de Grado y un proyecto de Investigación Operativa.

---

## Proyectos

### [TFG — Modelos Compartimentales Aplicados a la Dinámica del Crimen](./tfg-compartmental-models-crime-dynamics/)

**Asignatura:** Trabajo de Fin de Grado

Modelización matemática de la dinámica del crimen mediante modelos compartimentales de tipo SIR, ampliamente usados en epidemiología y adaptados aquí al contexto sociológico. El trabajo incluye:

- Formulación y análisis de sistemas de ecuaciones diferenciales ordinarias (EDO)
- Ajuste de parámetros a datos reales mediante técnicas de optimización
- Análisis del número reproductivo básico R₀ y estabilidad del sistema
- Simulaciones numéricas y validación del modelo

**Tecnologías:** Python, SciPy (resolución numérica de EDOs), Matplotlib

---

### [Investigación Operativa — GRASP y GRASP + Path Relinking para el MDP](./grasp-path-relinking-mdp/)

**Asignatura:** Investigación Operativa

Implementación desde cero de metaheurísticas para resolver el **Problema de Máxima Diversidad (MDP)**, un problema de optimización combinatoria NP-duro. El proyecto incluye:

- Implementación de **GRASP** (Greedy Randomized Adaptive Search Procedure): fase constructiva y búsqueda local
- Implementación de **GRASP + Path Relinking**: mejora mediante recombinación de soluciones élite
- Experimentación con instancias benchmark estándar (MDG-a)
- Comparativa de resultados entre ambas variantes

**Tecnologías:** Python (orientado a objetos), instancias MDG-a
