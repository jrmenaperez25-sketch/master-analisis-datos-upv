# Analisis Multivariante

Asignatura del Master en Analisis de Datos centrada en tecnicas de analisis multivariante.

## Contenido

- `Códigos/`: notebooks y codigo de trabajo.
- `medical_insurance.csv`: dataset utilizado en el analisis.
- `Trabajo Análisis Multivariante.pdf`: memoria final.
- `Vídeo AM.mp4`: defensa o presentacion del proyecto.

## Enfoque

El proyecto trabaja reduccion de dimensionalidad, analisis discriminante y tecnicas multivariantes sobre datos reales.
